# Мини-проект: Счетчик кликов / очков

Функции:
- `+1` увеличивает значение
- `-1` уменьшает значение
- `Сброс` возвращает значение к `0`

## Запуск

Откройте `index.html` в браузере.

Если браузер блокирует некоторые вещи при открытии файла напрямую, можно запустить простой локальный сервер:

```zsh
cd /Users/makarovmn/Public/js_project1/click-counter
python3 -m http.server 5500
```

Затем откройте:
- http://localhost:5500

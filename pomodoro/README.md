# Мини-проект: Таймер / Pomodoro (упрощённый)

Функции:
- `Старт` — запускает таймер (каждую секунду +1)
- `Стоп` — останавливает таймер
- Отображается текущее число секунд
- На 25-й минуте (1500 секунд) — сигнал через `alert()`

## Запуск

Откройте `index.html` в браузере.

Если нужно через локальный сервер:

```zsh
cd /Users/makarovmn/Public/js_project1/pomodoro
python3 -m http.server 5501
```

Открыть:
- http://localhost:5501

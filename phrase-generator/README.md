# Мини-проект: Генератор случайных фраз

Функционал:
- Кнопка «Сгенерировать»
- На экране появляется случайная фраза (мотивация / шутка / совет программисту)

## Запуск

Откройте `index.html` в браузере.

Либо через простой сервер:

```zsh
cd /Users/makarovmn/Public/js_project1/phrase-generator
python3 -m http.server 5502
```

Откройте:
- http://127.0.0.1:5502
